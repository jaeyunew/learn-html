    반응형 디자인 무엇인지?

    반응형 웹이라고 합니다. 
    - 기기 화면에 맞춰 요소가 자동으로 배치된 웹 페이지를 
      제공하므로 기기화면마다 따로 코드를 개발 관리하지 않아도 되며 유지관리도 용이함.

배치속성이 뭐야? 
CSS 레이아웃 

모든 HTML요소에는 display 속성이 있다.
display 속성의 기본값은 블럭 또는 인라인 설정되어 있다.

블록레벨 요소는 요소(태그)가 가로 줄 전체를 차지한다. <h1>
인라인레벨 요소는 요소가 있는 공간만을 차지한다. <span>

태그마다 이 두특성은 고정일까요? 
display 라는 CSS 속성으로 바꿔 줄 수 있다. 

블록과 인라인이라는 개념은 왜 필요할까?
보기 좋게 배치하기 위해서 필요하다. 
원하는 위치에 요소를 배치하려면 태그마다 가지고 있는 특성을 알아야한다. 

블럭과 인라인 요소 그리고 레이아웃을 정하는 HTML 요소가 바로 박스모델이다.

CSS 박스 모델은 기본적으로 모든 HTML을 감싸는 상자이다. 
- 요소(content) : 텍스트, 사진 등 보여줄 대상 
- 패딩(padding) : 요소 주변 영역을 감싼다. 해당 부분은 투명색이다. 
- 테두리 (border): 요소와 패딩을 감싸는 테두리이다. 색상 변경 가능
- 마진(margin) : 테두리 밖의 영역을 감싼다. 투명색이다.


(실습) 반응형 웹 페이지 
 미디어 쿼리(media query)를 이용하여 반응형 웹 페이지를 만든다. 
 사이트에 접속하는 미디어 타입과 특징에 따라 다른 
 CSS 스타일을 적용하도록 도와주는 규격이다. 

 반응형 웹 : 웹 사이트에 접속하는 디바이스의 해상도에 따라 HTML과
            CSS가 바뀌는 웹 

            - CSS에 @media @import 사용하여 특정 조건에 따라 스타일 적용
            - HTML 요소에 media 특성을 사용하여 특정 매체만 가리킬 때 사용

            @media (속성) {적용값} 
            @import 다른 스타일 시트의 규칙을 가져올때 사용한다. 

            @import url("hello.css") => 최상단에 기술해야 한다.
