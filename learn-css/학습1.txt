- CSS가 왜 필요한가?
  Cascading Style Sheet
  HTML의 역할은 정보를 표현
  CSS의 역할은 HTML 문서를 시각적으로 꾸미는 역할

- CSS 적용방법
  (1) HTML 문서 <style> CSS 코드 </style>
      웹브라우저(크롬) 한줄 한줄 해석 => 인터프리터 방식
      이 Style 태그 안쪽의 코드는 css 니까 css 문법에 맞도록 해석하고 처리해줘
  (2) 
- 외부 CSS 파일 적용

- 글씨 크기, 정렬

- CSS 선택자 (selector)
  html에 스타일을 적용할때
  html 요소를 지정하는 역할을 한다.

  내가 작성한 모든 태그에 스타일을 적용하려면
  일일이 태그를 적어야 하는가?

  - 전체 선택자 *
  - 유형 선택자 h1 p span
  - 아이디 선택자 : 해당 식별된 해그에 대해서만 골라서 적용

  - 클래스 선택자 : 다중 적용
    특정 클래스에 속하는 요소를 선택
    클래스란 요소의 목록을 의미한다.
    여러태그를 하나의 클래스로 묶을 수도 있고
    태그 하나에 여러 클래스 이름을 붙일 수도 있e다.

    클래스명과 일치하는 요소를 선택하고 하나 이상 스타일을 지정할 수 있다.

  - 복합 선택자
    부모와 자식태그가 복합적으로 쓰인다.
    (서로)의 관계와 위치를 유용하게 결합하는 방식을 제공한다.
    - 자손 선택자 : 조상의 모든 자손을 선택하는 방식
    - 자식 선택자
    - 인접 형제 선택자
    - 일반 형제 선택자
