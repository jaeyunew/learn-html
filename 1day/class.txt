코딩 vs 프로그래밍
코딩 : 컴퓨터에게 명령을 작성하는 일 자체를 의미 (코드를 작성하는 일)
프로그래밍
코드작성, 분석 및 구현, 디버깅, 컴파일, 테스트

웹페이지 vs 웹사이트
웹페이지 => Hyper Text로 작성된 문서
웹사이트 => URL 기반으로 인터넷에서 서비스되는 웹페이지들의 집합

URL => (uniform Resource locator) 네트워크에서 자원의 위치를 알려주는 규칙

1. 클라이언트 / 서버 구조에서 클라이언트는 요청(request) 서버는 응답(response)
2. 브라우저는 하이퍼텍스트를 해석하여 보여주는 클라이언트
3. HTML은 하이퍼텍스트를 표현하는 마크업 언어이다.
   웹페이지를 구조적으로 기술한다.
4. 하이퍼텍스트는 정해진 순서 없이 참조를 통하여 다른 문서에 접근할 수 있다.
5. 웹페이지는 웹에 있는 HTML로 작성된 문서이다.
6. 웹사이트는 URL 기반으로 인터넷에서 서비스되는 웹페이지들의 집합이다.
7. www(world wide web) = 웹 (www3c.org)
